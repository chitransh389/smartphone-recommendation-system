# 📱 Smartphone Recommendation System

## 📌 Description
This project analyzes smartphone data and recommends the best phone based on user preferences like budget, RAM, and features.

## 🚀 Features
- Data filtering (price, RAM, brand)
- Top smartphone recommendations
- Graphical analysis
- Smart recommendation system

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## ▶️ How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
